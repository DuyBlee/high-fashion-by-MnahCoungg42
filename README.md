## Giới thiệu dự án

Đây là dự án thiết kế website thời trang được xây dựng bằng HTML, CSS và một chút JavaScript.
Dự án được thực hiện trong 2 tuần, tập trung vào thiết kế trang web và thiết kế từng quần áo một.

## Tính năng

* Thanh điều hướng (navigation bar)
* Bố cục lưới sản phẩm (product grid)
* **MỖI SẢN PHẨM ĐỀU CÓ TRANG WEB RIÊNG**
* Tính năng "thêm vào giỏ hàng" sử dụng JavaScript
* Giao diện tối giản phong cách thời trang
* Footer

## Mục tiêu dự án
Mục tiêu của dự án là học cách xây dựng một website hoàn chỉnh từ đầu, hiểu cách bố trí layout, sử dụng Flexbox/Grid, thiết kế giao diện và tổ chức cấu trúc website như một trang thương mại điện tử thời trang thực tế.
